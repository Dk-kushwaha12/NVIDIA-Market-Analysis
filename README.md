# NVIDIA-Market-Analysis
📊 NVIDIA Market Analysis (EDA Project)
This project performs Exploratory Data Analysis (EDA) on NVIDIA market data to uncover trends, patterns, and insights that can support data-driven decisions.

🔹 Objective
Analyze historical stock performance of NVIDIA
Identify trends, volatility, and growth patterns
Extract insights useful for business and financial decision-making

🔹 Dataset
Source: Kaggle

🔹 Tools & Technologies
Python
Pandas
NumPy
Matplotlib / Seaborn
Jupyter Notebook

🔹 Key Steps Performed
Data Cleaning (handling missing values, formatting dates)
Exploratory Data Analysis (EDA)
Trend Analysis using time series plots
Moving averages to identify patterns
Correlation analysis

🔹 Visualizations
Line plots for trends
Box plot
Bar chart
Correlation heatmaps

 ┣ 📄 README.md
